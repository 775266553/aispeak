# 目录结构说明

项目按“主程序、测试、文档、资源”分层组织，便于后续维护和扩展。

```text
aispeak/
├─ start.bat                # GUI 启动入口
├─ start_web.bat            # Web 启动入口
├─ scripts/                 # 主程序与辅助脚本
├─ stitch/                  # Web 端后端与前端资源
├─ tests/                   # 测试与验收脚本
├─ doc/                     # 环境、使用、更新说明
├─ videos/                  # 示例音视频与转写输出
└─ skills/                  # 工具技能相关资源
```

约定如下：

- `scripts/` 放主程序脚本和通用工具脚本，不再混放测试文件。
- `tests/` 只放测试、烟雾测试和验证脚本。
- `doc/` 集中放环境要求、使用说明、更新说明和结构说明。
- `videos/` 作为示例输入和输出目录。
