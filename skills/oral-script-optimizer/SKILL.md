# Oral Script Optimizer

Turn source material into original, audience-friendly Chinese spoken scripts.

This skill is for situations where the user gives you one or more of the following:
- An article or long-form text
- Bullet-point notes or rough information
- One or more reference scripts they like
- A desired duration such as 3 minutes, 5 minutes, 8 minutes, or 10 minutes
- A tone request such as sharp, calm, story-like, analytical, emotional, or conversational

Your job is to preserve the useful properties of strong口播稿, then rewrite the content into a new script that sounds natural when spoken.

## Core principle

Use the reference material as a source of style and structure, not as text to copy.

Allowed:
- Near-synonym rewriting when it improves fluency
- Small amounts of shared paragraph order when the flow is genuinely useful
- Borrowing opening hooks, pacing patterns, contrast moves, and conclusion style
- Compressing or expanding sections to fit the requested duration
- Reorganizing examples, adding fresh examples, and changing the angle
- Using close-but-not-identical sentence rhythm when it makes the spoken version smoother
- Borrowing a similar paragraph sequence when the audience benefits from that progression, as long as the wording and examples are clearly new

Not allowed:
- Copying long phrases or consecutive sentences from the source
- Replacing only a few words while keeping the same sentence skeleton throughout
- Keeping the original logic too intact when the task asks for a new稿
- Producing text that would obviously be recognized as a lightly edited version of the source

When in doubt, make the piece feel more like a new script than a rewritten article.

## What makes the reference scripts work

The reference materials in this workspace share a few strong patterns. Use them as design signals.

### 1. They open with a big frame, not a small fact

These scripts usually do not begin with a narrow detail. They start by answering a larger question:
- What changed this year?
- Why does this event matter beyond the headline?
- What bigger structure is moving underneath the news?

That means the opening should quickly elevate the topic from “what happened” to “why it matters.”

### 2. They use contrast to create momentum

A common move is to set up a clean contrast such as:
- 不是某一件事大，而是整个系统在变
- 不是某个国家突然失控，而是旧秩序的收益和成本不再匹配
- 不是单点新闻，而是结构性转向

This kind of contrast gives the listener an immediate thesis and helps the script feel decisive.

### 3. They move from global to local to personal

A strong pattern in the samples is:
- world order / international system
- major regions or countries
- the user’s own country or audience-relevant situation
- future implications and personal stakes

This layered movement makes the script feel large in scope without losing human relevance.

### 4. They keep repeating the main thesis in different forms

The best口播稿 do not say the core point once and move on. They restate it with new words and new examples so the audience remembers it.

That repetition should feel like reinforcement, not copy-paste.

### 5. They end by returning to the viewer

Many strong reference scripts end by shifting from analysis back to the audience:
- what this means for us
- what we should watch next
- what kind of future is opening up

This makes the ending feel complete instead of merely informative.

## Style patterns to emulate

When the user asks for a script inspired by these materials, favor these patterns:

- Start with a thesis line that sounds confident and easy to remember.
- Use one or two short sentences to define the main conflict.
- Build the body in clean blocks, each block answering one question.
- Use transitional phrases that signal a shift in scale or topic.
- Keep spoken language tighter than written language.
- Put the strongest point early, then add evidence.
- Use rhetorical questions sparingly to reset attention.
- End with a forward-looking line that gives the listener a reason to care.

## Useful sentence patterns

Use these as inspiration, but rewrite them for the topic at hand:

- “这件事表面上看是X，实际上是在说明Y。”
- “我们真正要看的，不是A，而是A背后的B。”
- “如果只看新闻标题，你会以为这是一个孤立事件；但如果把它放回整个局势里看，它其实是一个信号。”
- “这不是结束，而是下一阶段的开始。”
- “你会发现，变化不是突然发生的，它只是终于被看见了。”

## Example transformation patterns

### Example 1: from article to 3-minute口播

Input idea:
- 一篇关于某个国际事件的长文

Good transformation:
- First line: one sentence that says why the event matters now
- Middle: 2 to 3 tightly packed points
- Ending: one short forecast or takeaway

What to avoid:
- Repeating the article’s section order line by line
- Preserving the article’s full background timeline
- Keeping too many written-style clauses

### Example 2: from reference scripts to new script

Input idea:
- Several excellent long口播稿 with a similar style

Good transformation:
- Extract the shared methods: strong opening, worldview framing, layered argument, final return
- Write a new thesis around the user’s topic
- Rebuild the examples so they fit the new subject
- Use a new sequence if it helps the flow feel more natural

What to avoid:
- Reusing the same analogies from the source too often
- Carrying over the original labels or named concepts when they are not necessary
- Keeping the same paragraph count just to look similar

## Reference style extraction checklist

Before writing, identify these elements from the source material:

1. Opening type
   - Thesis-first
   - Question-first
   - Scene-first
   - Story-first

2. Main logic type
   - Problem -> cause -> consequence
   - Past -> present -> future
   - Global -> regional -> local
   - Event -> pattern -> implication

3. Rhythm type
   - Fast and dense
   - Moderate with clear turns
   - Slow build with a powerful ending

4. Tone type
   - Sharp and critical
   - Calm and explanatory
   - Emotional and narrative
   - Strategic and analytical

5. Ending type
   - Summary
   - Prediction
   - Call to action
   - Reflective close

## Practical rewrite rule

If a sentence from the source feels worth keeping, do not preserve it directly. Instead, do one of the following:
- change the angle
- change the example
- change the order of information
- change the sentence length
- change the emotional color

If two or more of those changes are not happening, the draft is probably too close to the source.

## Recommended workflow

1. Identify the user goal
   - Is the user asking for a rewrite, a summary, a stronger hook, or a complete new script?
   - Determine the target duration and audience.

2. Analyze the source material
   - Extract the core viewpoint, key facts, tension points, and emotional arc.
   - Identify what makes the reference scripts work:
     - opening hook
     - pacing
     - transition style
     - paragraph rhythm
     - contrast and suspense
     - ending payoff

3. Choose a rewriting strategy
   - If the source is already strong, preserve the overall flow but re-express it.
   - If the source is weak or scattered, rebuild the logic from scratch.
   - If the user wants only inspiration, keep the structure looser and more original.

4. Rebuild the script
   - Write for the ear, not the eye.
   - Prefer short-to-medium spoken sentences.
   - Put the key point early.
   - Use transitions that guide listeners forward.
   - Make every section earn its place.

5. Fit the target duration
   - 3 minutes: sharp opening, one main thread, fast turns, tight conclusion.
   - 5 minutes: clearer setup, 2–3 supporting points, moderate buildup.
   - 10 minutes: fuller background, layered analysis, richer examples, slower release of tension.

6. Sanity-check originality
   - Compare the draft against the source mentally.
   - Replace any phrase that still feels too close.
   - Change examples, sequencing, or framing if needed.

## Duration guide

Default to short-form first unless the user clearly asks for a long-form deep dive.

Use these rough targets unless the user specifies otherwise:
- 3 minutes: about 450–700 Chinese characters, depending on speaking speed
- 5 minutes: about 800–1200 Chinese characters
- 10 minutes: about 1500–2200 Chinese characters
- 15 minutes: about 2200–3200 Chinese characters, better for layered expansion, more examples, and a more complete ending
- 30 minutes: about 4000–6000 Chinese characters, better for deep analysis, richer transitions, and a more fully developed closing

For most user requests, prioritize 3-minute and 5-minute scripts because they are faster to produce and better suited to short-video publishing.

Adjust for tone and pacing. Faster, denser评论类口播 can be shorter in characters; narrative or explanatory scripts may need more.

Longer scripts are best when the material needs:
- more background context
- more than one supporting example
- a gradual build instead of a fast punchline
- a fuller conclusion that lands the implication clearly

## Short-video priority

For short-video scripts, the main goal is not just correctness. It is retention.

Write 3-minute and 5-minute scripts so they feel:
- fast-paced
- eye-catching in the first few lines
- rich in hooks and turning points
- easy to keep listening to
- structured around one clear main thread

Think in terms of “continuous attention capture”:
- open strongly
- make the next sentence feel necessary
- keep the audience expecting the next turn
- avoid long flat stretches
- land a clean, memorable ending

In short-video mode, each paragraph should either:
- reveal a new layer
- sharpen the central conflict
- add a vivid example
- or move the listener closer to the conclusion

If a section does not help retention, shorten it or remove it.

## Output modes

Default to one polished script unless the user asks for variants.

If the user wants options, provide:
- Version A: more犀利 / 更抓人
- Version B: more稳重 / 更易播
- Version C: more故事感 / 更情绪化

## Output structure

When generating a script, use this structure unless the user asks otherwise:
1. Title
2. Hook or opening line
3. Main body in short spoken paragraphs
4. Transition lines between major ideas
5. Strong ending or closing takeaway

If useful, add a short note section after the script:
- Recommended speaking speed
- Places to pause or emphasize
- Optional cut points for video editing

## Script templates by duration

### 3-minute script

Use this shape:
1. Hook: 1-2 lines that define the conflict
2. Core thesis: 1 line that states the takeaway
3. Body: 2 compact sections with one supporting idea each
4. End: 1 short conclusion or forecast

This version should feel sharp and efficient. Do not spend too long on background. Treat it like a fast-moving short video: every sentence should either抓人、推进、或收束.

Use 3 minutes when the user wants:
- a quick opinion piece
- a fast comment on current events
- a highly concentrated hook-driven script
- the simplest possible structure with the strongest attention grab

For 3 minutes, the biggest rule is: get to the point fast and do not dilute the opening.

### 5-minute script

Use this shape:
1. Hook: a strong opener that frames the problem
2. Thesis: one clear sentence about the main takeaway
3. Body: 3 compact sections with one job each
4. Pivot: a transition that raises the stakes or widens the angle
5. End: a clean conclusion that lands the point without over-explaining

This version is still short-video friendly, but it can afford a little more development than 3 minutes. Use it when the user wants a script that still moves fast but has slightly more structure, contrast, and example support.

Use 5 minutes when the user wants:
- a short commentary with more room to explain
- a video script that still feels punchy but not rushed
- one extra layer of context or comparison
- a smoother rise toward the ending

For 5 minutes, the biggest rule is: keep the speed, but let the argument breathe just enough to feel fuller.

### 10-minute script

Use this shape:
1. Hook: a larger frame that opens the scene
2. Background: why the issue matters now
3. Core analysis: 3-5 sections, each with one clear job
4. Comparison: contrast with another region, period, or case
5. Implication: what this means for the future
6. Closing: return to the listener and land the point

This version is still short-video friendly, but it can breathe a little more than 3 minutes. Keep the rhythm tight and make sure there is a clear turn or highlight in almost every section.

This version can breathe more, but still needs spoken momentum. Every section should move the listener forward.

### 15-minute script

Use this shape when the user wants a more complete专题稿 or deeper analysis:
1. Hook: frame the issue in a broader way
2. Background: explain the context that makes the topic worth discussing
3. Core analysis: 4-6 sections with clear transitions
4. Supporting cases: add examples, comparisons, or small story beats
5. Implication: explain what changes for the audience or the future
6. Closing: leave the listener with a clearer judgment or takeaway

This format is ideal for “layer by layer” writing, because it gives enough room to unfold the argument without rushing the conclusion.

### 30-minute script

Use this shape when the user wants a long-form commentary, deep-dive narrative, or a complete video script:
1. Hook: open with a strong thesis or a scene-setting question
2. Framing: establish the big picture and why the topic matters
3. Sections 1-3: unpack the main line of argument
4. Sections 4-6: add more examples, counterpoints, or case studies
5. Synthesis: connect the details back to the larger pattern
6. Closing: end with a complete summary and a forward-looking judgment

This format works best for内容 that needs more room to develop, more cases to prove the point, and a more complete ending that does not feel rushed.

## Example output styles

### Style A: sharp commentary

Use when the user wants a stronger point of view.
- opening is decisive
- sentences are short
- transitions are punchy
- the ending lands hard

Example feel:
- “这不是偶然，这是一个系统开始换挡。”
- “你看到的是新闻，我看到的是结构。”

### Style B: analysis with story sense

Use when the user wants the script to be easier to follow.
- opening sets up a scene
- body uses a clear progression
- examples are vivid but not excessive
- ending ties the story back to a larger meaning

Example feel:
- “我们先别急着下结论，先看它为什么会在这个时间点发生。”
- “真正重要的，不是这个事件本身，而是它暴露出来的变化。”

### Style C: calm, authoritative explanation

Use when the user wants a more稳重的稿子.
- opening is less aggressive
- the argument unfolds steadily
- examples are practical
- the ending feels measured and credible

Example feel:
- “如果把它放回更大的背景里看，答案其实更清楚。”
- “这件事提醒我们，变化已经不是预期，而是现实。”

## Quality checklist

Before finalizing, confirm the script:
- Sounds natural when read aloud
- Has a strong opening in the first 1–3 lines
- Maintains a clear thread without drifting
- Avoids obvious source copying
- Fits the requested duration
- Uses fresh wording and fresh examples where possible
- Preserves the value of the reference scripts without imitating them too closely
- Sounds like a script someone can actually speak without stumbling
- Avoids too much article-style nesting or abstract stacking

## How to respond to the user

If the user provides reference scripts, briefly tell them what you learned from the style before presenting the new稿.

A useful response pattern is:
- “我先提炼它们的节奏、钩子和推进方式，再给你一个新的原创版本。”
- “这个版本会保留它的口播感，但会换掉表达、例子和局部结构。”

## Notes on originality

This skill should help the user borrow the strengths of优秀口播稿 responsibly.

Treat close paraphrase as a fallback only when the user explicitly wants a low-risk rewrite of factual material, and even then change the structure enough that the result still reads like a new script.

If the source text is itself very short or very generic, create a more original structure rather than staying close to it.
